CSS設定情報

ファイル構成

  • ソース: resources/css/app.css
  • ビルド: public/build/assets/app-[hash].css
  • 設定: vite.config.js
  • テンプレート: resources/views/layouts/app.blade.php

ビルドコマンド

  • 開発: npm run dev
  • 本番: npm run build
  • 監視: npm run watch

カスタマイズ方法

  1. resources/css/app.css でクラスを編集
  2. npm run build でCSSをリビルド
  3. ブラウザでリロードして変更を確認
  4. 必要に応じて新しいタイプやサイズを追加
